# 程序介绍
希望有个平台可以集中展示不同云平台上的主机，可以按照项目对主机分类‘   
并展示近一个月来主机资源的使用情况，依次判断是否有那个项目可以缩减资源，节约资金   

# 程序主要功能
1、单个和批量添加主机，页面展示主机   
2、根据添加的主机自动获取主机信息   
3、根据添加的主机自动获取主机硬件使用情况并形成图表，可以按照时间范围选择每分钟更新一次，类似zabbix监控  
4、使用django的计划任务优化数据库数据，比如针对10天前的数据就无需每分钟一个点了，例如1小时一个点，减少数据库里的数据       
5、按照项目展示项目下的主机  
6、用户管理，登录、退出、修改密码  
7、用户组管理
